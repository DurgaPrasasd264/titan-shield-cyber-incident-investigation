# Titan Shield Cyber Incident Investigation

## Project Overview
This repository documents a SOC-style cyber incident investigation
conducted in a simulated enterprise environment named Titan Shield.

The investigation focuses on phishing-based initial access, malware
execution, data exfiltration, and nation-state threat actor activity
using Microsoft Defender XDR and KQL.

## Tools & Technologies
- Microsoft Defender XDR
- Kusto Query Language (KQL)
- Threat Intelligence
- MITRE ATT&CK Framework

## Skills Demonstrated
- SOC alert investigation
- Email and phishing analysis
- Endpoint process investigation
- Threat actor attribution
- Incident response documentation

## Disclaimer
This project is based on an educational lab scenario and is intended
strictly for learning and professional portfolio purposes.
